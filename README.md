# Зубной таймер

MVP Android-приложения для детского таймера чистки зубов.

## Что уже есть

- Главный экран
- Запуск чистки в одно нажатие
- Последовательный таймер по зонам
- Пауза между зонами
- Пауза/продолжение чистки
- Досрочное завершение с записью в историю
- Экран истории
- Экран настроек
- Редактирование списка зон
- Изменение времени каждой зоны
- Изменение порядка зон
- Добавление и удаление зон
- Сброс к настройкам по умолчанию

## Зоны по умолчанию

1. Жевательная поверхность — верх справа — 20 сек
2. Жевательная поверхность — верх слева — 20 сек
3. Жевательная поверхность — низ справа — 20 сек
4. Жевательная поверхность — низ слева — 20 сек
5. Внешняя боковая поверхность — верх справа — 10 сек
6. Внешняя боковая поверхность — верх слева — 10 сек
7. Внешняя боковая поверхность — низ справа — 10 сек
8. Внешняя боковая поверхность — низ слева — 10 сек
9. Передние зубы — сверху — 15 сек
10. Передние зубы — снизу — 15 сек

## Сборка

Проект собирается в GitHub Actions. После загрузки файлов в репозиторий нужно открыть вкладку Actions, дождаться завершения сборки и скачать артефакт `toothbrush-timer-debug-apk`.
